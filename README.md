## EEG ADHD

This project prepares and structures EEG data for ADHD analysis.

## Project files (Current)


- `EegDataset.py`
  - Defines the `EegDataset` dataclass.
  - Stores dataset-level metadata such as:
    - `name`
    - `sampling_freq_hz`
    - `participants`
    - `num_channels`
    - `adhd_num`
    - `control_num`
    - `channel_names`
    - `raw_data_path`
    - `interim_data_path`
    - `metadata_path`

- `nasarbadi_builder.py`
  - Loads raw ADHD EEG data from `./datasets/raw_data/adhdata.csv`.
  - Converts the combined raw dataset into per-subject CSV files under `./datasets/interim_data/nasarbadi/`.
  - Builds a metadata index CSV at `./datasets/metadata/nasarbadi_subject_index.csv` containing:
    - subject ID
    - duration in seconds
    - class label
    - number of samples
  - Uses the `EegDataset` dataclass to store dataset metadata before generating files.

## Dataset structure

- `datasets/raw_data/adhdata.csv`
  - The original raw dataset containing all EEG recordings and subject labels.

- `datasets/interim_data/nasarbadi/`
  - Per-subject CSV exports generated by `nasarbadi_builder.py`.

- `datasets/metadata/nasarbadi_subject_index.csv`
  - Generated subject index metadata for later analysis.

## Notes

- The builder script assumes a sampling frequency of `128 Hz` and `19` EEG channels.
- The raw dataset uses subject IDs in the `ID` column and class labels in the `Class` column.
